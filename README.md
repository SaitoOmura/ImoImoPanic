# ImoImoPanic
作物を育てながら敵を撃退し、制限時間内に収穫してスコアを競う農園ディフェンスアクションゲームです。

## 使用技術
- C++
- DxLib

## 制作人数
6人

## 担当範囲
※チーム制作のため、自分が担当した部分のみ記載しています

- ゲームメインループ
- プレイヤー操作（移動・攻撃）
- 敵（モグラ・イノシシ）の挙動制御
- スコア・リソース管理
- ゲーム進行に応じたイベント制御

## 技術的なポイント
- 時間やスコアに応じて状態が変化するゲーム進行管理
- 敵出現やイベント発生を制御する仕組みの実装
- コントローラー振動によるフィードバック処理

## 工夫した点
ゲーム終盤に向けて難易度や環境が変化する仕組みを実装し、プレイヤーの判断が重要になるよう設計しました。  
また、振動機能を活用し、触覚によるフィードバックを追加することで没入感を向上させています。

## 学んだこと
早い段階で基礎機能を完成させることで、ブラッシュアップに時間を使えることを学びました。
